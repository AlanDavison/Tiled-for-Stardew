Methods

addLayer

  • addLayer(layer: Layer): void

  • Adds the layer to the group, above all existing layers. The layer can't already be part of a map.

    When adding a {@see TileLayer} to a map, the layer's width and height are automatically initialized to the size of the map (since Tiled 1.4.2).

    Parameters

    • layer: Layer

    Returns void
